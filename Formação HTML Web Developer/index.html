O que é o HTML?

HTML (HyperText Markup Language) é a linguagem de marcação padrão usada para criar páginas da web. É a estrutura básica que define a estrutura e o conteúdo de uma página da web.

O HTML consiste em uma série de elementos ou tags que são usados para marcar diferentes partes do conteúdo, como cabeçalhos, parágrafos, imagens, links, listas e muito mais.

Ferramentas utilizadas

IDE (Integrated Development Environment) é um software que fornece um ambiente integrado para o desenvolvimento de software. Ela é projetada para facilitar a programação, oferecendo recursos como edição de código, depuração, compilação, gerenciamento de projetos e muitos outros recursos úteis para desenvolvedores.

O Visual Studio Code (VSCode) é uma das IDEs mais populares e amplamente utilizadas atualmente. Ele é desenvolvido pela Microsoft e é um editor de código-fonte abrangente e altamente personalizável.

O VSCode suporta uma ampla gama de linguagens de programação e oferece recursos avançados, como realce de sintaxe, depuração integrada, controle de versão, terminal embutido, extensões personalizáveis e uma comunidade ativa de desenvolvedores que contribuem com diversas extensões para estender ainda mais as funcionalidades da IDE.

Inspetor de Elementos

O "Inspetor de Elementos" do Google é uma ferramenta integrada ao navegador Google Chrome, que permite visualizar e inspecionar os elementos de uma página da web em tempo real. É uma parte da suíte de ferramentas de desenvolvimento do Chrome, destinada a ajudar desenvolvedores e designers a entender e depurar o código HTML, CSS e JavaScript de uma página da web.

Para acessar o Inspetor de Elementos no Google Chrome, você pode clicar com o botão direito do mouse em qualquer elemento de uma página da web e selecionar a opção "Inspecionar" no menu de contexto. Você também pode acessá-lo usando o atalho de teclado Ctrl+Shift+I (ou Command+Option+I no macOS).

O que são tags?

As tags HTML são elementos ou marcadores usados para marcar diferentes partes de um documento HTML e definir sua estrutura e conteúdo. As tags são escritas entre colchetes angulares ("<" e ">") e geralmente consistem em um nome de tag que descreve o tipo de elemento que está sendo marcado.

Existem diferentes tipos de tags HTML, cada uma com um propósito específico. Algumas das tags mais comuns incluem:

  1. Tags de cabeçalho: 'h1','h2','h3','h4','h5','h6'. São usadas para criar cabeçalhos ou títulos em diferentes níveis de importância.
  2. Tags de parágrafo: 'p'. Marcam um parágrafo de texto.Tags de parágrafo: 'p'. Marcam um parágrafo de texto.
  3. Tags de links: 'a'. São usadas para criar links para outras páginas ou recursos.
  4. Tags de imagem: 'img'. São usadas para inserir imagens na página.
  5. Tags de lista: 'ul', 'ol', 'li'. Permitem criar listas não ordenadas (com marcadores) e listas ordenadas (numeradas).

O que são artibutos?

Em HTML, os atributos são adicionados às tags para fornecer informações adicionais sobre um elemento específico. Eles são usados para modificar o comportamento ou a aparência do elemento. Os atributos são definidos dentro da tag de abertura de um elemento e consistem em um nome e um valor, separados por um sinal de igual (=).

Aqui estão algumas características importantes dos atributos em HTML:

  1. Nome do atributo: É o nome que define a propriedade que será modificada ou especificada. Por exemplo, o atributo "href" é usado na tag 'a' para definir o URL do link.
  2. Valor do atributo: É o valor associado ao atributo. Ele fornece informações específicas ou configurações para o atributo. Por exemplo, no atributo "href", o valor seria o URL da página para a qual o link aponta.
  3. Sintaxe: A sintaxe geral para adicionar um atributo é 'nome_do_atributo="valor_do_atributo"'. Por exemplo, 'a href="https://www.exemplo.com"'.
